## 🚀 What is CineMatch?

CineMatch is a **Netflix-style movie recommendation system** that:

- 🎯 Recommends movies using **collaborative filtering**
- 🎬 Lets users **select profiles (like Netflix)**
- ⭐ Allows users to **add watched movies + ratings**
- 🧠 Updates recommendations based on **your watch history**
- 🎨 Features a **cinematic UI inspired by Netflix**

---

## ✨ Features

- 🔐 Login screen (UI-based)
- 👤 Profile selection (Person 1, 2, 3)
- 🎯 Personalized recommendations
- 🔥 Popular movie discovery
- 🎭 Mood-based filtering (Action, Romance, etc.)
- 🎬 Genre-based filtering
- ➕ Add watched movies + rating
- ⚡ Real-time recommendation updates
- 🎨 Fully styled cinematic UI

---

## 🧠 How It Works

1. Load MovieLens dataset  
2. Create **user-movie matrix**
3. Compute **movie similarity using cosine similarity**
4. Recommend movies based on:
   - user history
   - similar users/items
5. Apply filters:
   - mood
   - genres
6. Display results in UI
